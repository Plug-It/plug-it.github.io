About Plug-It

The history

Plug-It started out as a theme that I (WiBla) made for myself.
I often shared my progress to others in the community, and some were really interested in using it as well.
I then developed a quick way for everyone to load it using a bookmarklet (bookmark script). It was ok, but you had to click it every time and it became tideous after a while..

Once I was happy with my theme, I wondered if I could add other features using JavaScript and the provided API. Which I did! The first couple of features included the already famous "Auto-Woot" and Hide Video!
Time passed and I was adding features and improving on my theme on a daily basis.
After some time, I searched for a way to auto-load the script whenever you would browse plug.dj. The ideal solution were to create a Chrome Extension so I spent time documenting myself on the subject and start creating my own.
It is now available for everyone to use!

It's also a community

Plug-It is made by and for Plug.dj users, I take feedback very seriously and work hard to provide you with the best I can.

The goals

Plug-It strives in providing you with customisation as well as moderation features that fits your needs!

It's open-source!

If you are as curious as I am, you probably want to get a look under the hood and understand how everything works. Or maybe you want to make sure that I don't steal any of your precious data?! In any case, the Github is here for you!
